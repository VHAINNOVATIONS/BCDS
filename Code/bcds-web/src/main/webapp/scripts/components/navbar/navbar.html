• Veteran ID: Unique Identifier for each Veteran/Customer.
    • Veteran Name: Name of Veteran (Derived value for pilot).
    • Claim ID: Unique Identifier for each claim filed by the Veteran/Customer.
    • Model: The name and version of the model applied by the user.
    • Model Result ID: Unique identifier for the result generated by the model.
    • Prior Rating: The prior rating (or CDD) included in the matched pattern.
    • Rater Evaluation: The rating produced by the RVSR as a result of the manual adjudication process.
    • Model Results: The rating generated by the model (i.e., contained in the matched pattern).
    • RE/MR Match: Indication as to whether the modeled rating is equal to the actual rating.
    • Pattern Rate of Use: The number of times the matched pattern has occurred within the last 8 years.
    • Pattern Accuracy: The number of times the matched pattern as resulted in the same rating as a fraction of the number of times it has occurred within the last 8 years.
    • Agree (Y/N): Rater indicates if he/she agrees with the Model output/result.
